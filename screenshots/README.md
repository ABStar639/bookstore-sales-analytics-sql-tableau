# Screenshots

This folder contains visual outputs from the Book Store Sales Analytics project.

## Dashboard

The Tableau dashboard provides an interactive view of key business metrics and performance indicators, including:

* Revenue by Genre
* Top Customers by Revenue
* Revenue by Country
* Inventory Health Analysis

### Purpose

The dashboard was designed to help business stakeholders:

* Identify high-performing genres and revenue drivers
* Understand customer purchasing behavior
* Monitor country-level sales performance
* Assess inventory health and stock availability
* Support data-driven decision-making
