# Dataset

This project uses three datasets representing a bookstore's operations.

## Files

### Books.csv
Contains information about books including:
- Book ID
- Title
- Author
- Genre
- Price
- Stock
- Published Year

### Customers.csv
Contains customer information including:
- Customer ID
- Name
- Email
- Country
- City

### Orders.csv
Contains transactional sales data including:
- Order ID
- Customer ID
- Book ID
- Quantity
- Order Date
- Total Amount

These datasets were used to analyze customer behavior, revenue generation, inventory status, and genre-level performance.
